The Problem

Traditional EEG signal processing relies on manual feature extraction, which is both time-consuming and prone to errors. Moreover, existing algorithms often focus on specific frequency bands or waveforms, neglecting the rich interplay between different components of the EEG signal. To make matters worse, many research institutions still rely on outdated hardware and software architectures that can't keep pace with the demands of modern neuroscience.

For instance, a study published in the Journal of Neuroscience Methods highlighted the limitations of traditional frequency-based analysis methods when applied to EEG data (Kaiser et al., 2020). The researchers found that these approaches failed to capture subtle differences between subjects, leading to inconsistent results. This is where our AI-powered EEG analysis platform comes into play.

import numpy as np
from liam_core import BinaryAlgebra


EEG signal processing with vectorized operations

algebra = BinaryAlgebra()
raw_eeg = np.random.randn(64, 256)  # 64 channels, 256 samples

Bandpass filter via FFT (vectorized)

fft = np.fft.rfft(raw_eeg, axis=1)
freqs = np.fft.rfftfreq(256, 1/256)
mask = (freqs >= 0.5) & (freqs <= 45)
filtered = np.fft.irfft(fft * mask, axis=1)

Feature extraction with matrix operations

alpha_power = algebra.frobenius_norm(filtered[:, 8:13])
print(f"Alpha band power: {alpha_power:.2f}")
