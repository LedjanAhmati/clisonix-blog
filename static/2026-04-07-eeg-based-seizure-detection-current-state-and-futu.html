The Problem

EEG (electroencephalogram) signals provide a wealth of information about brain activity, but processing and interpreting these signals is no trivial task. Seizure detection requires identifying patterns amidst the complex, high-dimensional data generated by EEG recordings. Even with advanced signal processing techniques, subtle variations in brain activity can be missed, leading to false positives or false negatives.

Moreover, current seizure detection algorithms often rely on machine learning models trained on limited datasets, which may not generalize well to real-world scenarios. These limitations highlight the need for more robust and adaptive methods that can cope with the variability inherent in EEG signals.

import numpy as np
from liam_core import BinaryAlgebra


EEG signal processing with vectorized operations

algebra = BinaryAlgebra()
raw_eeg = np.random.randn(64, 256)  # 64 channels, 256 samples

Bandpass filter via FFT (vectorized)

fft = np.fft.rfft(raw_eeg, axis=1)
freqs = np.fft.rfftfreq(256, 1/256)
mask = (freqs >= 0.5) & (freqs <= 45)
filtered = np.fft.irfft(fft * mask, axis=1)

Feature extraction with matrix operations

alpha_power = algebra.frobenius_norm(filtered[:, 8:13])
print(f"Alpha band power: {alpha_power:.2f}")
