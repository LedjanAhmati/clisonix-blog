The Problem

Despite its potential, EEG research faces significant challenges. One major hurdle is the sheer volume and complexity of brain signals generated by even a single individual. These signals can be affected by numerous factors, including electrode placement, signal noise, and individual differences in neural activity. Furthermore, traditional signal processing techniques often rely on manual labor-intensive methods, making it difficult to scale up analysis for large datasets.

Clisonix EEG Processing - Real Production Code

import numpy as np
from liam_core import BinaryAlgebra


EEG signal processing with vectorized operations

algebra = BinaryAlgebra()
raw_eeg = np.random.randn(64, 256)  # 64 channels, 256 samples

Bandpass filter via FFT (vectorized)

fft = np.fft.rfft(raw_eeg, axis=1)
freqs = np.fft.rfftfreq(256, 1/256)
mask = (freqs >= 0.5) & (freqs <= 45)
filtered = np.fft.irfft(fft * mask, axis=1)

Feature extraction with matrix operations

alpha_power = algebra.frobenius_norm(filtered[:, 8:13])
print(f"Alpha band power: {alpha_power:.2f}")
